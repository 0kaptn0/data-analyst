Airline Flight Analysis Dashboard
This project is a Dash application that visualizes the total number of flights to different destination states in the U.S. based on airline reporting. It uses Plotly for interactive visualizations and Pandas for data manipulation.

Project Overview
The dashboard displays a bar plot that shows the total number of flights to each destination state, allowing the user to filter the data by year.

Technologies Used:
Pandas: For data manipulation and filtering.
Plotly: For interactive data visualizations (specifically Plotly Express and Plotly Graph Objects).
Dash: For building the web application and rendering dynamic visualizations based on user input.
How It Works
Data Source:
The application uses data from the IBM Developer Skills Network airline dataset, which contains flight-related information, including destination states and the number of flights per state for different years.

Dataset URL:
Airline Data CSV

App Features:
Input Year: Users can input a specific year to filter the data (default: 2010).
Bar Plot: Displays a bar chart showing the total number of flights to destination states for the selected year.
